إذا كان لديك PocketBase يعمل على جهازك وتريد أن يصل إليه تطبيق Flutter من الإنترنت، فيمكنك استخدام Cloudflare عبر خدمة Cloudflare Tunnel.
هذه الطريقة تجعل جهازك يعمل كسيرفر بدون فتح منافذ في الراوتر. 🚀

سأشرح لك الطريقة خطوة بخطوة.

1️⃣ تشغيل PocketBase على جهازك

2️⃣ إنشاء حساب في Cloudflare

اذهب إلى موقع:

أضف دومين (مثل example.com).

⚠️ ملاحظة: يجب أن يكون لديك دومين.

3️⃣ تثبيت Cloudflare Tunnel

قم بتحميل cloudflared من الموقع:

https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/install-and-setup/

ثم ثبته على جهازك.

4️⃣ تسجيل الدخول إلى Cloudflare

افتح Terminal أو CMD واكتب:

cloudflared tunnel login

سيفتح المتصفح لتسجيل الدخول.

5️⃣ إنشاء Tunnel

اكتب:

cloudflared tunnel create pocketbase

سيتم إنشاء Tunnel خاص بك.

6️⃣ ربط الدومين مع السيرفر

اكتب:

cloudflared tunnel route dns pocketbase api.example.com

الآن أصبح لديك رابط مثل:

https://api.example.com
7️⃣ ربط Tunnel مع PocketBase

شغل Tunnel:

cloudflared tunnel --url http://localhost:8090

الآن أي شخص يفتح:

https://api.example.com

سيصل إلى PocketBase على جهازك.

8️⃣ استخدامه داخل Flutter

في تطبيق Flutter ضع رابط API:

final pb = PocketBase('https://api.example.com');

مثال تسجيل مستخدم:

await pb.collection('users').authWithPassword(
  'test@gmail.com',
  '12345678',
);

أنشئ ملف باسم:

Dockerfile

واكتب فيه:

FROM alpine:latest

WORKDIR /app

RUN apk add --no-cache unzip wget

RUN wget https://github.com/pocketbase/pocketbase/releases/download/v0.22.0/pocketbase_0.22.0_linux_amd64.zip

RUN unzip pocketbase_0.22.0_linux_amd64.zip

EXPOSE 8090

CMD ["./pocketbase", "serve", "--http=0.0.0.0:8090"]

docker build -t pocketbase-server .

1️⃣ إنشاء حساب Oracle Cloud مجاني

اذهب إلى:

https://www.oracle.com/cloud/free/

اضغط على “Start for free”

أنشئ حساب جديد باستخدام بريد إلكتروني صالح

أدخل بطاقة بنكية للتأكيد (لن يتم الخصم)

بعد التسجيل، ستحصل على VPS مجاني مدى الحياة ضمن خطة Always Free.

2️⃣ إنشاء جهاز افتراضي (Compute Instance)

بعد تسجيل الدخول:

اذهب إلى Compute → Instances → Create Instance

اختر نظام التشغيل: Ubuntu 22.04

اختر حجم الجهاز المجاني VM.Standard.E2.1.Micro

1 CPU

1 GB RAM

اترك باقي الإعدادات كما هي واضغط Create

⚠️ بعد بضع دقائق سيظهر لك IP الجهاز.

3️⃣ فتح البورت لـ PocketBase

لتتمكن من الوصول إلى PocketBase عبر الإنترنت:

اذهب إلى Networking → Virtual Cloud Network → Security Lists

أضف قاعدة جديدة:

Type: TCP

Port Range: 8090

Source: 0.0.0.0/0

4️⃣ الاتصال بالسيرفر عبر SSH

إذا كنت على Windows، استخدم PuTTY أو Git Bash:

ssh ubuntu@IP_ADDRESS

استبدل IP_ADDRESS بعنوان VPS الخاص بك.

5️⃣ تثبيت PocketBase

على السيرفر اكتب:

sudo apt update
sudo apt install unzip wget -y
wget https://github.com/pocketbase/pocketbase/releases/download/v0.22.0/pocketbase_0.22.0_linux_amd64.zip
unzip pocketbase_0.22.0_linux_amd64.zip
6️⃣ تشغيل PocketBase

لتشغيل السيرفر على البورت 8090:

./pocketbase serve --http=0.0.0.0:8090

الآن افتح المتصفح:

http://IP_ADDRESS:8090/_/

ستظهر لوحة تحكم PocketBase.

7️⃣ ربطه مع تطبيق Flutter

في Flutter ضع الرابط:

final pb = PocketBase('http://IP_ADDRESS:8090');

الآن تطبيقك يستطيع الوصول إلى السيرفر 24 ساعة مجاناً.

تمام! سأشرح لك طريقة تشغيل PocketBase على Oracle VPS مع HTTPS وربطه مباشرة بتطبيق Flutter خطوة بخطوة، بحيث يصبح تطبيقك جاهز للاختبار والنشر الاحترافي. 🚀

1️⃣ تثبيت PocketBase على Oracle VPS

اتصل بالسيرفر عبر SSH:

ssh ubuntu@IP_ADDRESS

حدّث النظام وثبت الأدوات:

sudo apt update && sudo apt upgrade -y
sudo apt install unzip wget -y

حمل PocketBase:

wget https://github.com/pocketbase/pocketbase/releases/download/v0.22.0/pocketbase_0.22.0_linux_amd64.zip
unzip pocketbase_0.22.0_linux_amd64.zip

شغل PocketBase على البورت 8090:

./pocketbase serve --http=0.0.0.0:8090

الآن يمكنك فتح في المتصفح:

http://IP_ADDRESS:8090/_/
2️⃣ إضافة HTTPS باستخدام Nginx وCertbot

تثبيت Nginx:

sudo apt install nginx -y

تثبيت Certbot:

sudo apt install certbot python3-certbot-nginx -y

إعداد Nginx ليكون Reverse Proxy:

sudo nano /etc/nginx/sites-available/pocketbase

ضع هذا الكود (استبدل yourdomain.com بالدومين الخاص بك):

server {
    listen 80;
    server_name yourdomain.com;

    location / {
        proxy_pass http://127.0.0.1:8090;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}

فعل الموقع وأعد تشغيل Nginx:

sudo ln -s /etc/nginx/sites-available/pocketbase /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx

احصل على شهادة SSL مجانية:

sudo certbot --nginx -d yourdomain.com

اتبع التعليمات حتى ينتهي، الآن لديك HTTPS تلقائي.

3️⃣ ربط Flutter مع السيرفر

في تطبيق Flutter ضع الرابط الآمن:

final pb = PocketBase('https://yourdomain.com');
مثال تسجيل مستخدم:
await pb.collection('users').authWithPassword(
  'test@gmail.com',
  '12345678',
);

يعمل عبر HTTPS

4️⃣ نصائح مهمة

لتشغيل PocketBase تلقائياً بعد إعادة التشغيل، استخدم systemd:

sudo nano /etc/systemd/system/pocketbase.service

ضع الكود:

[Unit]
Description=PocketBase Server
After=network.target

[Service]
User=ubuntu
WorkingDirectory=/home/ubuntu
ExecStart=/home/ubuntu/pocketbase serve --http=0.0.0.0:8090
Restart=always

[Install]
WantedBy=multi-user.target

ثم:

sudo systemctl enable pocketbase
sudo systemctl start pocketbase

الآن PocketBase سيعمل تلقائياً 24 ساعة.
